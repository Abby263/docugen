## Getting Started

### Registration

1. Visit the XunLong SaaS homepage
2. Click "Get Started" or "Sign Up"
3. Enter your email, username, and password
4. Click "Create Account"

You'll start with a **Free** tier account (5 documents per month).

### First Login

1. Go to the login page
2. Enter your username/email and password
3. Click "Sign In"

You'll be redirected to your dashboard.

## Creating Your First Project

### Step 1: Navigate to Project Creation

- Click "New Project" button (top right)
- Or go to Dashboard → Projects → New Project

### Step 2: Fill in Project Details

#### Project Title
Give your project a descriptive name:
- ✅ "AI in Healthcare 2024"
- ✅ "Market Analysis - Electric Vehicles"
- ❌ "My Project" (too generic)

#### Description (Optional)
Add context about what you want to generate.

#### Research Query ⭐ Most Important
Be specific and detailed:

**Good Examples:**
```
"Latest developments in artificial intelligence for healthcare 
diagnostics, including machine learning applications, FDA approvals, 
and implementation challenges in hospitals"
```

```
"Comprehensive analysis of the electric vehicle market in Europe, 
focusing on Tesla, VW, and BYD market share, charging infrastructure 
development, and government policies from 2022-2024"
```

**Tips:**
- Include specific topics you want covered
- Mention time periods if relevant
- Specify geographical scope if applicable
- List key entities, companies, or technologies

### Step 3: Choose Document Type

#### Report (General)
- Comprehensive overview of any topic
- Balanced coverage of multiple aspects
- Best for: General research, summaries

#### Analysis
- In-depth analytical perspective
- Data-driven insights
- Best for: Business analysis, trend analysis

#### Research
- Academic-style documentation
- Detailed methodology and citations
- Best for: Academic research, technical topics

#### Daily Brief
- Quick summary format
- Highlights key points
- Best for: News summaries, quick updates

#### Presentation (PPT)
- Multi-slide HTML presentation
- Visual layout with navigation
- Best for: Presentations, pitch decks

### Step 4: Select Output Format

- **HTML**: Web-friendly, styled, can be viewed in browser
- **Markdown**: Plain text, easy to edit, portable

### Step 5: Create Project

Click "Create Project" - your document generation will start automatically!

## Document Generation Process

### What Happens After Creation

1. **Pending** (0%): Project queued
2. **Processing** (0-100%): AI is working
   - Searching the web
   - Analyzing results
   - Generating content
   - Formatting document
3. **Completed** (100%): Ready to download!

### Real-Time Updates

- Progress bar shows current status
- WebSocket provides live updates
- Average time: 30-120 seconds

### Viewing Results

When completed, you can:
- **Preview**: View in browser
- **Download**: Save to your computer
- **Share**: Copy link (coming soon)

## Best Practices

### Writing Effective Queries

#### Be Specific
❌ "AI in business"
✅ "AI applications in retail inventory management, including case studies from Walmart and Amazon, ROI analysis, and implementation challenges"

#### Include Context
❌ "Climate change"
✅ "Impact of climate change on agriculture in Sub-Saharan Africa, focusing on drought patterns, crop yields, and adaptation strategies from 2020-2024"

#### Mention Key Elements
- Who: Companies, people, organizations
- What: Technologies, products, events
- When: Time periods, dates
- Where: Geographic locations
- Why: Goals, motivations
- How: Methods, processes

### Document Type Selection

| Use Case | Recommended Type |
|----------|-----------------|
| Research project | Research |
| Business proposal | Report |
| Market study | Analysis |
| News summary | Daily Brief |
| Meeting presentation | PPT |
| Product overview | Report |
| Trend analysis | Analysis |

### Optimizing Results

1. **Review and refine**: Download and review your first document
2. **Adjust query**: If results aren't focused enough, make query more specific
3. **Try different types**: Same query with different document types gives different perspectives
4. **Use descriptions**: Add context in description field

## FAQ

### How long does document generation take?

Typically 30-120 seconds, depending on:
- Query complexity
- Document type
- Current server load

### Can I edit generated documents?

Yes! Download as Markdown format for easy editing.

### What languages are supported?

Currently English. Support for other languages coming soon.

### Is my data secure?

Yes. All data is:
- Encrypted in transit (HTTPS)
- Stored securely
- Not shared with third parties
- Backed up regularly

### Can I cancel during processing?

Yes, click "Cancel" on the project detail page.

### What if generation fails?

- Check the error message
- Verify your query is clear
- Try rephrasing your query
- Contact support if issues persist

### How accurate are the documents?

Documents are generated using:
- Multiple web sources
- AI synthesis and analysis
- Up-to-date information

Always review and verify critical information.

### Can I get a refund?

Pro tier offers 14-day money-back guarantee.

### Is there an API?

Yes! Pro and Enterprise tiers include API access.
See API documentation for details.

### How do I get support?

- Free tier: Community forum
- Pro tier: Email support (24-48h response)
- Enterprise: Dedicated support manager

### Can I collaborate with team members?

Team collaboration is available on Enterprise tier.
